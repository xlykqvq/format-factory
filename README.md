# 格式工厂（推荐面简历生成器）

一个用于**硕士推免/推荐面**的简历生成网页：左侧实时编辑，右侧 A4 简历预览，并支持导出 PDF。

## 功能

- **实时修改内容**：输入即预览；数据会自动保存到浏览器本地（刷新不丢）
- **导出 PDF**
  - **一键导出 PDF**：使用页面右上角按钮导出
  - **打印/另存为 PDF**：也可用浏览器打印功能保存为 PDF
- **内容模块**
  - 教育背景
  - 科研经历
  - 竞赛经历
  - 自我评价
- **填写提醒**：自动提示还缺哪些关键项、还能加哪些内容（技能/项目/产出/链接等）

## 启动

需要 Node.js（建议 18+）。

```bash
npm install
npm run dev
```

浏览器打开终端提示的本地地址即可。

## 构建

```bash
npm run build
npm run preview
```
